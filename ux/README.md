# UX Team Workspace

This directory contains a shared structure for UX workstreams in the `ai-learning` repository.

## Folder structure

- `research/`
  - `plans/` - research plans, protocols, and recruiting briefs.
  - `findings/` - synthesized insights, reports, and readouts.
  - `personas/` - personas, journey maps, and audience artifacts.
- `design/`
  - `wireframes/` - early screen/layout explorations.
  - `ui-kit/` - reusable styles, tokens, and design patterns.
  - `components/` - component-level design references.
- `prototypes/`
  - `low-fidelity/` - quick concept prototypes.
  - `high-fidelity/` - polished interactive prototypes.
- `content/`
  - `copy-guidelines/` - voice/tone and writing conventions.
  - `microcopy/` - in-product text drafts.
- `assets/`
  - `icons/` - icon exports and source files.
  - `images/` - image assets used in UX artifacts.
  - `illustrations/` - custom illustration assets.
- `handoff/`
  - `specs/` - implementation-ready product specs.
  - `redlines/` - spacing/size/interaction redline docs.
- `testing/`
  - `usability/` - test scripts and study outputs.
  - `accessibility/` - accessibility checks and audits.
- `meetings/`
  - `notes/` - UX sync notes
